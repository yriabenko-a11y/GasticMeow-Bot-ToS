## 🔒 Privacy Policy

1. **Data Collection:** We collect User IDs, Server IDs, and basic configuration data to provide bot functionality.
2. **Dashboard Access:** When using the dashboard, we access your Discord profile and guild list via OAuth2 to manage settings. We do not store your credentials.
3. **Usage:** Data is used strictly for bot features (Economy, Levels, Moderation). We never sell or share data with third parties.
4. **Data Deletion:** To request your data be deleted, please contact the developer or join the support server.
